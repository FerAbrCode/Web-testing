## Accessible Interactive Photo Gallery

This project is a small, accessible, interactive photo gallery that supports:

- Mouse interaction (hover)
- Keyboard interaction (tab + focus, plus Enter/Space)
- Dynamic preview updates with accessible text alternatives

### Run locally

- Open `index.html` in a browser.

### How to use

- Hover over a gallery tile to update the preview.
- Press `Tab` to move focus through tiles; the preview updates on focus.

### Validation checklist

- HTML: https://validator.w3.org/
- WAVE: https://wave.webaim.org/
- aXe (browser extension): https://www.deque.com/axe/browser-extensions/

### Hosting (example: GitHub Pages)

- Push this folder to a GitHub repo.
- In repo settings → **Pages** → deploy from the `main` branch (root).
- Share the Pages URL.
